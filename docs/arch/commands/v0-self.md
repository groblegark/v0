# v0-self

**Purpose:** Self-management commands dispatcher.

## Usage

```bash
v0 self update              # Update to latest stable
v0 self update nightly      # Switch to nightly
v0 self version             # Show version info
v0 self debug <operation>   # Generate debug report
```

See: [v0-self-update](v0-self-update.md), [v0-self-version](v0-self-version.md), [v0-self-debug](v0-self-debug.md)
