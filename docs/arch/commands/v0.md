# v0

**Purpose:** Main CLI entry point and command dispatcher.

## Workflow

1. Parse command from arguments
2. Handle special commands (`init`, `help`, `version`)
3. Check command type and dispatch to `v0-<command>`

## Usage

```bash
v0 init                              # Initialize project
v0 feature auth "Add JWT auth"       # Full autonomous pipeline
v0 fix "Button not working"          # Report a bug
v0 status                            # List all operations
```

## Aliases

`feat`→`feature`, `decomp`→`decompose`, `bug`→`fix`, `chat`→`talk`, `resume`→`feature --resume`
